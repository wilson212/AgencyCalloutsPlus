Agency Dispatch and Callouts+ Scenario List

Description:
	Each callout will have multiple scenarios. Each scenario is essentially
	a full callout all in itself. Each scenario will have multiple dialogs
	that are chosen at random, so playing the same scenario multiple times
	will still feel different each time (ex: different reasons for crime, 
	different attitude and responses).

File Format:
 - Callout Name
	- Scenario Name
